# Communication Management Plan

## 1. Objective
Define how communication will be organized within a team of 18 people during the project, establishing clear channels, responsibilities, and usage rules to ensure effective and structured communication.

## 2. Team organizational structure
The team is organized as follows:
- **Project Leader:** overall coordination, decision-making, and external contact.
- **Area Managers:** supervision of specific tasks and channeling team questions.
- **Team Members:** execution of assigned tasks and reporting issues.

## 3. General communication rules
- Keep messages clear and concise.
- Use mentions (@) only when necessary.
- Respond to important messages within a maximum of 24 hours.
- Prioritize official channels over private messages.
- Avoid unnecessary message overload.

---

## 4. Communication via Microsoft Teams

### 4.1 Purpose of using Teams
Microsoft Teams will be the **main communication and coordination tool** for the project.

### 4.2 Teams structure
- **Team:** Project
- **Channels:**
  - **General:** official announcements and important decisions.
  - **Project management:** planning, task tracking, and progress monitoring.
  - **Daily work:** operational questions and daily coordination.
  - **Documentation:** file management and version control.
  - **Social / Informal:** non-project-related communication.

### 4.3 Who to contact depending on the situation (Teams)
| Situation              | Contact person                | Channel            |
| ---------------------- | ----------------------------- | ------------------ |
| Task-related questions | Area Manager                  | Daily work         |
| Deadline issues        | Area Manager / Project Leader | Project management |
| Important changes      | Project Leader                | General            |
| Documents and versions | Documentation Manager         | Documentation      |
| Conflicts or blockers  | Project Leader                | Private chat       |

### 4.4 Specific rules for using Teams
- Official announcements must be published only in the **General** channel.
- Meetings will be held via Teams and summarized in writing.
- Avoid duplicating information in other channels.

---

## 5. Communication via WhatsApp

### 5.1 Purpose of using WhatsApp
WhatsApp will be used as a **secondary communication channel**, only for fast or urgent communications.

### 5.2 Allowed use of WhatsApp
- Urgent project-related notices.
- Quick reminders about meetings or imminent deadlines.
- Fast coordination in exceptional situations.

### 5.3 Non-allowed use of WhatsApp
- Making important decisions.
- Sending official documentation.
- Long discussions or task tracking.

### 5.4 Specific rules for using WhatsApp
- WhatsApp does not replace Microsoft Teams.
- Any relevant information must be later transferred to Teams.
- Avoid messages outside the agreed working hours, except in urgent cases.

---

## 6. Monitoring and improvement
This communication management plan may be reviewed and adjusted during the project if communication issues or improvement needs are identified.
