# Development Tools (Early Phase)

This document lists **tools and applications** used during the development process of the project.

## 1. Project Management & Organization

Tools to plan, prioritize, and track progress:

* **Github issues** – Issue tracking, epics, sprints, roadmap
* **Github projects** – Kanban Boards are very useful for organization purposes.
---

## 2. Design & UX/UI

Tools for designing and validating the user experience:

* **Canva** – Used for the creation of mockups
* **** - Used for the creation of UML diagrams

---

## 3. Team Communication

Tools to keep the team aligned:

* **Whatsapp** – Daily communication, team channels
* **Microsoft Teams** – Meetings, reviews

---

## 4. Version Control & Collaboration

Core collaboration tools for development:

* **GitHub** – Repositories, pull requests, issues
* **GitHub Issues / GitHub Projects** – Code-linked task tracking
* **GitKraken / Sourcetree** – Visual Git clients (optional)

---

## 5. Development & Personal Productivity

Tools used daily by developers:

* **VS Code** – Code editor
* **EditorConfig** – Consistent formatting across the team
* **Prettier / ESLint (extensions)** – Code style and consistency
* **Todoist / TickTick** – Personal task management (optional)

---

## 6. Testing, QA & Validation

Tools for testing and validating functionality:

* **Postman / Insomnia** – Manual API testing
* **Swagger UI** – API exploration and validation
* **Playwright UI / Cypress Dashboard** – E2E test visualization
* **BrowserStack** – Cross-browser and device testing

---

## 7. DevOps, CI/CD & Environments

Tools to automate builds, tests, and deployments:

* **GitHub Actions / GitLab CI** – CI/CD pipelines
* **Docker Desktop** – Local development environments
* **Dependabot / Renovate** – Dependency update automation
* **Railway / Render Dashboards** – Deployment management (early-stage)

---

## 8. Knowledge Management & Decision Tracking

Tools to avoid losing context over time:

* **Markdown Docs (in-repo)** – Technical documentation
